# ToDoList_VanilaJS
프로그래머스 데브코스 
Vanila JS 역량 강화
강의를 들으면서 만들어본
컴포넌트 방식으로 구현한
ToDoList 입니다.

#component를 구성하기
1. main.js 를 최상위 컴포넌트로 구성(App.js를 호출)
2. App.js를 통해 다른 컴포넌트들과 연결
3. Header 영역
4. Form 영역
5. List 영역 
6. 으로 구분하여 컴포넌트를 구성 
7. LocalStorage를 통해 input value를 저장하고 불러오는 로직을 구현

#단순히 문자열을 입력하고, 리스트에 노출되는 기능을 구현함!!!
